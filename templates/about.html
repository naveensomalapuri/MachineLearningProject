StudMathPredict(GEEMM) About

Summary of the End-to-End Generic Machine Learning Project

  • In this generic ML project, we built an end-to-end pipeline using Python and Flask. It includes data ingestion, transformation, and model training. Various models were evaluated using GridSearchCV for hyperparameter tuning. The best model is saved for predictions. The web app allows users to input data and get predicted math scores. Tools used: Python, Git, GitHub, Anaconda, Jupyter Notebook, Flask, scikit-learn, pandas.
